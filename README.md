# CLI

Connect CLI to Claude, Codex, and ChatGPT through JoAi's hosted MCP app server.

[Open app page](https://joai.ai/en/apps/cli) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-cli@joai-claude-plugins
```

### Codex

Install this app from the official JoAi Codex marketplace in Codex.

`joai-cli`

If your Codex build only supports MCP registration, use the MCP endpoint below as a manual fallback.

### Cursor

Use the [Cursor install link](https://cursor.com/en-US/install-mcp?name=joai-cli&config=eyJ1cmwiOiJodHRwczovL2NvcnRleC5qb2FpLmFpL21jcC9hcHBzL2NsaSJ9) to install directly.

Or copy this fallback URL:

```
https://cortex.joai.ai/mcp/apps/cli
```


### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/cli
```
