# CLI

Connect CLI to Claude, Codex, and ChatGPT through JoAi's hosted MCP app server.

[Open app page](https://joai.ai/en/apps/cli) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-cli@joai-claude-plugins
```

### Codex

```bash
codex mcp add joai-cli --url https://cortex.joai.ai/mcp/apps/cli
```

### Cursor

Copy to local plugins:

```bash
git clone https://github.com/joai-plugins/cli.git ~/.cursor/plugins/local/joai-cli
```

Or use the [MCP deeplink](cursor://anysphere.cursor-deeplink/mcp/install?name=joai-cli&config=eyJtY3BTZXJ2ZXJzIjp7ImpvYWktY2xpIjp7InVybCI6Imh0dHBzOi8vY29ydGV4LmpvYWkuYWkvbWNwL2FwcHMvY2xpIn19fQ==) to install directly.

### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/cli
```
